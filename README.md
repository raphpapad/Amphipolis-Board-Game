The goal of the programming exercises was the implementation of the board game Amphipolis in electronic form (in Java)  in the strongest and best condition. The game is in 2-D, which was achieved with the graphics. Then inheritance, polymorphism, and encapsulation are used. Inheritance and polymorphism are used in the model, and to achieve encapsulation, the variables are made private. The game has three main parts. The first is the model, which is responsible for maintaining the current state of the game and the moves that the player will make. Secondly, there is the controller, who is responsible for the functionality and implementation of the game. Third, we have the View, which provides machine-user interaction.
